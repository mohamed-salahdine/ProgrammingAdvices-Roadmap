# 15 - Database Level 1 - SQL (Concepts and Practice)

Completed: February 2026

## Course Overview

- Full foundational SQL course from Programming Advices roadmap
- Covered: Database concepts, ERD & relational design, keys & constraints, DDL/DML, SELECT queries (filtering, joins, grouping, subqueries), Views, Indexes, Normalization (1NF–3NF+), basic performance considerations

## Key Learnings & Takeaways

- How to properly design entities and relationships using ER diagrams
- Importance of primary keys, foreign keys, and referential integrity
- Difference between various JOIN types (INNER, LEFT, RIGHT, FULL) and when to use each
- Normalization process: avoided redundancy and anomalies up to 3NF
- Creating and using VIEWs for simplified querying
- Basic INDEX usage for performance

## Challenges I Faced

- Getting comfortable with multi-table JOIN syntax (especially with 3+ tables)
- Understanding when to use GROUP BY + HAVING vs WHERE
- Designing good primary/foreign key strategies in complex schemas
